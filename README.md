# Todo App (React)

A modern and responsive Todo App built with React for managing daily tasks efficiently.  
It allows users to organize, track, and manage tasks with a clean and user-friendly interface.

---

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed / pending
- Filter tasks by status (All / Active / Completed)
- Responsive UI (Mobile + Desktop)

---

## Tech Stack

- React.js
- Tailwind CSS
- JavaScript (ES6+)

---

## Purpose

This project was built to practice React fundamentals such as state management, components, and handling user interactions in a real-world application.

---

## Live Demo

Try the app live : https://todo-app-react-liard-alpha.vercel.app/

## Installation

```bash
git clone https://github.com/Youssef-W-Bashir/todo-app-react.git
cd todo-app-react
npm install
npm run dev
```
