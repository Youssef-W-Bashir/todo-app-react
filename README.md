# Todo App (React)

A simple and clean Todo App built with React.  
It helps users manage their daily tasks efficiently with filtering and status tracking.

---

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed / pending
- Filter tasks by status (All / Active / Completed)
- Responsive UI

---

## Tech Stack

- React.js
- JavaScript (ES6+)
- TailwindCss

---

## Purpose

This project was built to practice React fundamentals like state management, components, and handling user interactions in a real-world Todo App.

---

## Installation

`bash
git clone https://github.com/Youssef-W-Bashir/todo-app-react.git
cd todo-app-react
npm install
npm start
